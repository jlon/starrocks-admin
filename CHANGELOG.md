# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [1.0.0] - 2024-12-05

### Added
- Complete internationalization (i18n) support for Chinese and English
- Profile visualization with DAG analysis
- Smart query optimization recommendations
- Kubernetes Helm chart support

### Changed
- Improved UI/UX across all pages
- Enhanced error handling and user feedback

### Fixed
- Various bug fixes and performance improvements

## [0.1.0] - 2024-12-05

### Added
- Initial release
- Multi-cluster management
- Real-time monitoring and metrics
- Query management and execution
- Node management (FE/BE)
- Session management
- Variable management
- Materialized view management
- Role-based access control (RBAC)
- Multi-tenant organization support
- Docker and Kubernetes deployment support
