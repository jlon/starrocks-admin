# 开发环境准备

## 安装Rust
'''powershell
wsl
sudo curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sudo sh -s -- -y

source ~/.cargo/env && cargo --version

export PATH="$HOME/.cargo/bin:$PATH" && cargo --version

# 缺少OpenSSL开发库
sudo apt update && wsl sudo apt install -y libssl-dev pkg-config
```
